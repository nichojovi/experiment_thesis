Welcome to number of staircase problem!

Task:
Write a function called print_number_staircase(start, step, n) that print the value of a single staircase using the formula:
staircase = start + step * n

Example:
If start = 2, step = 3, and n = 5, the output should be:
17

Hint: 
- Pay attention to the formula.
- Don't forget to print the result using print().
